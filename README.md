# bryantotty.com

Personal website - static HTML + Tailwind CSS.

## Local Preview

Just open `index.html` in a browser. No build step required.

## Deployment

This site can be deployed for free via:
- **GitHub Pages**: Settings → Pages → Deploy from branch `main`
- **Cloudflare Pages**: Connect the repo, no build command needed

## Structure

```
index.html     : Home page
about.html     : About me
blog.html      : Blog listing
blog/*.html    : Individual blog posts
projects.html  : Projects listing
contact.html   : Contact info
privacy.html   : Privacy policy / Terms of use
style.css      : Custom styles
main.js        : Mobile menu + scroll animations
```
